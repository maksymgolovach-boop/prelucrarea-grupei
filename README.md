# Prelucrarea grupei
## O aplicatie care permite prelucrarea datelor studentilor introdusi cu scrierea si citirea in/din fisier
Functionalitatea aplicatiei:
+ Introducere datelor studentilor
+ Afisarea formatata a datelor
+ Salvarea/citirea din fisier text
+ Afisarea studentilor promovati, notele si mediile lor
+ Afisarea numarului de studenti promovati
+ Afisarea medii generale a grupei la al doilea test
+ Crearea fisierului cu studenti picati
Aplicatia aceasta este un exemplu de o aplicatie programata pur pe notiunile programarii procedurale.
